What You'll Learn

Structured Curriculum

From basic strokes to finished compositions

Module 1

Introduction to calligraphy tools, basic pen angles, and foundational strokes. Understanding pressure and flow.

Module 2

Lowercase and uppercase alphabets in classic scripts (Copperplate / Italic). Letter spacing and consistency.

Module 3

Word formation, connecting letters, and basic flourishing techniques.

Module 4

Modern brush calligraphy – using brush pens, blending colors, and creating bounce lettering.

Module 5

Layout design, composition, and adding decorative elements. Creating greeting cards and quotes.

Module 6

Final project – complete artwork piece. Certificate and exhibition opportunity.
